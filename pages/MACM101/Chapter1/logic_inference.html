Definition 2. An argument is a series of statements(premises) and ending with a conclusion.

Definition 3. An argument valid if and only if when all the premises are true, the conclusion is necessarily true. In other words, an argument is valid if and only if it is equivalent to a tautology.

An argument is not valid if and only if it is not equivalent to a tautology.
